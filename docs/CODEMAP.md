- `scripts/e2e/smoke.mjs` ‚Äî the main 8-step scenario (~30s): join+ROOM_STATE ‚Üí **adopt self's position before the first move** (trap P1) ‚Üí walk at 12Hz to pocket-slime-south ‚Üí assert 0 corrections ‚Üí find a mob+cast ‚Üí assert MSG_SKILL_RESULT ¬∑ note: constants (MSG_*, speed, sync Hz, spawn, collision) are **copied from config/net-protocol with line-number comments** ‚Äî .mjs can't import .ts; if you change config/protocol, you must also update this file
- `scripts/e2e/README.md` ‚Äî how to run it locally/on prod (`npm run e2e` or `E2E_RT_URL=wss://... node scripts/e2e/smoke.mjs`) + cold-start caveats
